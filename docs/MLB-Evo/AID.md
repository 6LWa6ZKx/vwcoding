# Приборная панель

### Свободное место в баке

```
Блок 17 → Кодирование
> Байт 10 — Бит 4 → активировать
→ Применить (с перезагрузкой блока)
```
```
ODIS E: 006: 0017 → [LN]_refuel_volume: yes
```
